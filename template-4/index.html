Creative Technology

"

Creative Technology is not just about writing code or building circuits — it's about using technical tools imaginatively to solve problems that matter. It's where engineering precision meets human-centred thinking.

As a mechanical engineering student who also builds web tools, I sit at the intersection of physical systems and digital solutions. My goal is to bridge these two worlds to create tools that are practical, accessible, and impactful.

⚙️

Automation

Automating repetitive tasks — like manual college searching or PDF data entry — saves time and eliminates human error. That's the core promise of control systems engineering applied to software.

🌐

Web Tools

Browser-based tools need zero installation and work on any device. Making engineering utilities accessible through the web is democratising knowledge — anyone can use them.

🔩

Engineering Systems

Systems thinking — understanding how components interact — applies equally to mechanical assemblies and to code architecture. Every project I build follows a structured, systems-based approach.
